# <img src="./icons/icon48.png" width="48" align="absmiddle"> obookmark

一款专为 Microsoft Edge (及 Chromium 内核浏览器) 设计的轻量级书签同步扩展。

## 🌟 功能特点

- **手动触发同步**：完全自主掌控同步时机，点击即刻备份。
- **仅导出收藏夹栏**：与 Edge 官方导出行为一致，排除冗余目录，并自动重命名为 `bookmarks.html`。
- **安全防护**：
  - **隐私加密**：WebDAV 凭据在本地以混淆格式存储。
  - **安全锁屏**：UI 界面防窥视，禁止全选和复制敏感信息。
- **多语言支持**：自动适配简体中文与英文。
- **黑暗模式**：界面颜色随浏览器主题自动切换。
- **WebDAV 支持**：完美支持各种标准的 WebDAV 服务器。

## ⚠️ 同步原理与局限性

1. **非实时增量同步**：本项目并非类似官方 Edge 的实时同步，而是通过 WebDAV 手动将书签「全量导出」至云端，或在另一设备上手动「全量覆盖」本地书签。
2. **全量备份模式**：程序不具备变更检测功能，无法自动识别书签的增删改。每次同步均为完整的书签库导出。
3. **拉取功能限制**：当前版本仅实现了书签的上传备份功能，**暂不支持**直接从 WebDAV 服务器拉取书签并全量覆盖本地 Edge 浏览器收藏夹。

## 🚀 安装方法

1. 下载本项目或克隆到本地。
2. 打开 Edge 浏览器，进入 `edge://extensions/`。
3. 开启左下角的 **“开发人员模式”**。
4. 点击 **“解压缩后的扩展”**。
5. 选择本项目所在文件夹。

## 🛠️ 使用说明

1. 点击浏览器工具栏的 `obookmark` 图标。
2. 输入您的 WebDAV 服务器地址、用户名和密码。
3. 点击 **“保存设置”**。
4. 点击 **“立即同步书签”** 完成备份。
